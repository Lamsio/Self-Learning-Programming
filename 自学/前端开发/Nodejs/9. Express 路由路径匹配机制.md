## Express 路由路径匹配机制
#### 基本
```js
router.get('/',(req, res)=>{
	res.send("匹配成功");
})
```

#### 字符串符 ?
```js
// acd 或 abcd 匹配成功
router.get('/ab?cd',(req, res)=>{
	res.send("匹配成功");
})
```

```js
// abcde 或 ade 匹配成功
router.get('/a(bc)?de',(req, res)=>{
	res.send("匹配成功");
})
```

#### 字符串符 *
```js
// 匹配0个以上符合的字符
// 匹配: abcd abmcd abnsadasdasdasdasdacd...
router.get('/ab*cd',(req, res)=>{
	res.send("匹配成功");
})
```

#### 正则表达式
```js
// 匹配包含a的任意值
// 匹配: a abcd gdsagsdfsdf...
router.get(/a/,(req, res)=>{
	res.send("匹配成功");
})
```

具体请参考JavaScript正则表达式

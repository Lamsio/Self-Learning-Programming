## 组件
#### 非单文件组件
即，多个组件存在于同一个文件中，这种情况并不适用于日常开发，为了条理清晰，应当采用单文件组件，即`filename.vue`

#### Option data的两种编写方式
Option的data属性有两种编写方式，分别是`对象式`与`函数式`.

###### 对象式
```javascript
Vue.createApp({
        data : {
            a:1
        }
    })
```

###### 函数式
```js
const vm = Vue.createApp({
    data() {
        return {
            opacity: 1
        }
    },
}).mount('#app');
```

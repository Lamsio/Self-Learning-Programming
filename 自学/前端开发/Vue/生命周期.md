## 生命周期
#### 引出生命周期
当我们尝试实现让`<h2>小标题</h2>`产生渐变透明效果时，我们将需要用到`setInterval`，于是乎我们将会产生以下代码。
```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
 <meta charset="UTF-8">  
 <title>Title</title>  
</head>  
<body>  
	 <div id="app">  
		 <h2 :style="{opacity}">H2标题</h2>  
	 </div>
</body>  
  
<script src="https://unpkg.com/vue@next"></script>  
<script>  
 const vm = Vue.createApp({  
    data() {  
        return {  
            opacity: 1  
 			}  
    },  
 }).mount('#app');  
  
 // 通过外部定时器实现效果（不推荐）  
 setInterval(()=>{  
        vm.opacity -= 0.01;  
 if(vm.opacity <= 0) vm.opacity = 1;  
 },16)  
</script>  
</html>
```

#### Vue2的响应式原理
- 实现原理:
	- 对象类型: 通过`Object.defineProperty()`对属性的读取、修改进行拦截（数据劫持）
	- 数组类型: 通过重写更新数组的一系列方法来实现拦截

```js
Object.defineProperty(data, 'count', {
	get(){},
	set(){},
})	
```

- 存在问题:
	- 新增属性、删除属性，界面不会更新
	- 直接通过下标修改数组，界面不会自动更新

#### Vue2代码
```html
<template>
	<div>
		<h2>这里是Vue2展示区</h2>
		<p>姓名: {{person.name}}</p>
		<p>年龄: {{person.age}}</p>
		<p>性别: {{person.sex}}</p>
		<button @click="addSex()">增加性别</button>
	</div>
</template>

<script>
export default {
	name: 'App',
	data(){
		return {
			person: {
				name: '张三',
				age: 14,
			}
		}
	},
	methods: {
		addSex(){
			this.person.sex = "男"
		}
	}
}
</script>
```

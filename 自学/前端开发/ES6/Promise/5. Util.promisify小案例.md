## Util.promisify案例
#### 一. 更改文件展示方法
```txt
小明=90 小红=98 小兰=26 小刚=72
```

我们希望上面的文件内容变为:

```txt
小明=90  
小红=98  
小兰=26  
小刚=72
```

```js
let path = './marks.txt'  
const b = async function(){  
    let data = await newReadFile(path);  
	return data.toString().replaceAll(' ','\n');  
}().then(value => {  
    newWriteFile(path,value)  
}).catch(e=>{  
    console.log(e)  
})
```

## 单例模式

#### 饿汉式
思想：
1. 创建类的私有构造器
2. 创建类的私有对象属性
3. 创建类的公开获取方法

```java
public class SingletonClass {  
    private static SingletonClass sc = new SingletonClass();  
    private SingletonClass(){  
  
    }  
    public static SingletonClass getInstance(){  
        return sc;  
    }  
}
```

## 条件查询
1. QueryWrapper查询
```java
@Test  
void Query(){  
    QueryWrapper qw = new QueryWrapper();  
    qw.lt("id",3);  
    List<User> l = userDao.selectList(qw);  
    System.out.println(l);  
}
```

2. Lambda
```java
@Test  
void Query(){  
    QueryWrapper<User> qw = new QueryWrapper<User>();  
    qw.lambda().lt(User::getId,2);  
    List<User> l = userDao.selectList(qw);  
    System.out.println(l);  
}
```
这种方式是为了避免手动填写列名出错，如果你有十足把握不出错，也能这样写。

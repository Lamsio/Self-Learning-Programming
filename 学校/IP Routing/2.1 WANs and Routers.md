#### Introduction to Router
```text
RAM : Stores routing table, holds ARP cache, provide temporary memory for
configuration file. Data will be lost when a router is powered down.

NVRAM : Provides storage for the startup configuration file. 
Data will be kept if the router is powered down.

Flash : Hold the IOS image. Data will be kept if the router is powered down.

ROM : Store bootstrap program and basic OS software

Console : Provide physical access for initial configuration.

Interfaces : Connect routers to a network for packet entry or exit
```
